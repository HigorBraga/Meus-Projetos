Trabalhando com operadores Aritméticos.

+ = para Adição
- = para Subtração
* = para Multiplicação
/ = para Divisão
% = para Resto da divisão inteira
** = para Potência

Ordem de Precedência dos operadores Aritméticos

1° ()
2° **
3° * / %
4° + -

Ordem de Precedência dos operadores relacionais

1 = (>)
2 = ("<")
3 = (>=)

Na verdade não tem uma ordem de precedência faz quem aparecer primeiro

Ordem de precedência dos operadores lógicos

1° = ! (Operador de negação)
2° = && (Operador de junção)
3° = || (Operador de disjunção)

Agora a ordem de precedência completa com todos os operadores visto

1° os operadores aritméticos
2° os operadores relacionais
3° os operadores lógicos
